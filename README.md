# 🛒 E-Commerce Analysis Dashboard

## 📌 Project Overview

This project is a **JavaFX-based E-Commerce Analysis Dashboard** that allows users to upload a CSV file and visualize sales data using tables, KPIs, and charts.

It provides a modern, dark-themed analytics interface for better data understanding and presentation.

---

## ✨ Features

* 📂 Upload CSV file dynamically
* 📋 Display data in a table
* 💰 KPI Cards:

  * Total Sales
  * Total Profit
  * Total Orders
* 📊 Interactive Charts:

  * Bar Chart (Sales by Category)
  * Pie Chart (Category Distribution)
  * Line Chart (Sales Trend)
* 🌑 Dark Theme UI (Premium Look)
* 🎨 Modern and clean dashboard design

---

## 🛠️ Tech Stack

* **Java**
* **JavaFX**
* **CSV File Handling**

## ▶️ How to Run

1. Open terminal in project folder

2. Compile:

```
javac --module-path "path_to_javafx_lib" --add-modules javafx.controls,javafx.fxml *.java
```

3. Run:

```
java --module-path "path_to_javafx_lib" --add-modules javafx.controls,javafx.fxml Main
```

---

## 📊 How It Works

1. User uploads a CSV file
2. Data is parsed and stored
3. KPIs are calculated dynamically
4. Charts are generated using aggregated data
